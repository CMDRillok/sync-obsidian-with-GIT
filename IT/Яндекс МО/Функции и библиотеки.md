Вместо того, чтобы изобретать велосипед, ты можешь найти уже построенный — то есть, использовать готовый код в своей программе. Для этого в программировании есть **[[функция]]** и **[[библиотека]].**

Функции — это отдельные подпрограммы, которые ты можешь написать в одном месте, а потом использовать где угодно дальше в своем коде. Они записываются так:

```python
def название_функции(параметры через запятую):
    команды функции
    return данные, которые возвращает функция
```

После объявления функции ее можно вызывать: `название_функции(передаваемые параметры)`.

Библиотеки — это данные и функции, собранные в удобные тематические сборники. В начале программы нужно подключить библиотеку командой `import название_библиотеки`, и можно будет использовать все ее возможности.
