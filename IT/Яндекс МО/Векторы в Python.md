Так как [[вектор]] — это попросту набор чисел, то для его представления в Python больше всего подходит список. А точнее — более продвинутая версия списка: специальный тип данных массив из библиотеки **numpy**. Библиотека numpy реализует математические вычисления, так что все операции с векторами удобно делать с ее помощью.
Массив [[numpy]] создаётся с помощью функции `np.array()`, в которую передается **список Python**.

Такой код создает вектор (1, 2):
```python
import numpy as np # импортируем библиотеку

vector = np.array([1, 2]) # создаем вектор
print(vector) # печатаем результат
```

Вычисление величины (длины) вектора
```python
import numpy as np

vector = np.array([3, 4]) # создаем вектор
norm = np.linalg.norm(vector) # вычисляем его величину (норму)
print("Величина вектора:", norm) # выводим результат
```

Умножение на [[скаляр]] (число, [[Умножение числа на вектор]])
Для умножения на число особых функций не нужно — сработает обычная операция умножения. Так как вектор — это объект типа numpy-массив, то он воспримет умножение как умножение вектора на число.
```python
import numpy as np

v = np.array([2, 3, -1]) # вектор
scalar = 4 # скаляр
result = scalar * v # умножение вектора на скаляр

print(v) # выведет [ 2  3 -1]
print(result) # выведет [ 8 12 -4]
```

[[Сложение векторов]]
То же самое актуально и для сложения векторов:
```python
import numpy as np

# создаем два вектора
v1 = np.array([1, 1, -1])
v2 = np.array([-1, -1, -1])

result = v1 + v2 # складываем

print(result) # выведет [ 0  0 -2]
```

Скалярное произведение векторов([[Скалярное произведение векторов]])
Для операции скалярного произведения в библиотеке numpy есть функция `dot()`:
```python
import numpy as np

# создаем два вектора
v1 = np.array([1, -1])
v2 = np.array([10, 1300])

result = np.dot(v1, v2) # умножаем

print(result) # выведет -1290
```
