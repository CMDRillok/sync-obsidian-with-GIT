- Представь, что у тебя есть набор [[Объект]]ов с несколькими параметрами([[параметр]]). Например, ты исследуешь дворовых котиков, и для каждого котика записываешь два параметра: длина хвоста и скорость убегания от толпы любвеобильных детей.
- Цель твоего исследования — выяснить: помогают ли котикам длинные хвосты в побеге?
- Такую зависимость (или ее отсутствие) можно увидеть наглядно, если расставить котиков на графике, где по одной оси будут длины хвостов, а по другой — скорость бега. Если зависимость есть, точки выстроятся в понятную картинку. Если нет — будут случайно рассредоточены по полю.
Такая диаграмма называется точечной.
- [[Pandas]] [[Matplotlib]] [[Визуализация данных pandas, matplotlib]] [[данные]] [[значение]] [[график]]

```python
data = {
    "Количество зараженных файлов": [5, 0, 25, 0, 0, 7, 0, 0, 28, 0, 21, 12, 0, 0, 15, 27],
    "Средняя оценка": [4.3, 4.9, 3.2, 4.3, 4.7, 3.2, 2.7, 3.3, 3.1, 4.2, 3.2, 3.3, 3.1, 3.8, 3.2, 3.1]
}

df = pd.DataFrame(data)

plt.figure(figsize=(8, 5))
# точечная диаграмма
plt.scatter(df["Количество зараженных файлов"], df["Средняя оценка"], color="blue", edgecolor="black", alpha=0.7)

# оформление графика
plt.xlabel("Зараженные файлы")
plt.ylabel("Средняя оценка")
plt.title("Зависимость учебных результатов от встречи с вирусом")
plt.grid(True, linestyle="--", alpha=0.7)

# отображаем график
plt.show()
```
