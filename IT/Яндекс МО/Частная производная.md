Частная производная — это производная функции по одному ее параметру. Остальные параметры функции при вычислении воспринимаются как константы.
- В Python частную производную посчитать так же легко, как и обычную: [[sympy]]

```python
import sympy as sp

# создаем переменные
x, y = sp.symbols('x y')

# задаем функцию
f = x**2 * y + 3*x*y**2

# частная производная по x
df_dx = sp.diff(f, x)

# частная производная по y
df_dy = sp.diff(f, y)

print("Δf/Δx =", df_dx) # напечатает: Δf/Δx = 2*x*y + 3*y**2
print("Δf/Δy =", df_dy) # напечатает: Δf/Δy = x**2 + 6*x*y
```
