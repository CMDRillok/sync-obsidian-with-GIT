- это основной [[тип данных]] в библиотеке [[Pandas]]. Он выглядит как [[таблица]]!
- [[библиотека]], [[Функции и библиотеки]]
```python
# подключаем библиотеку
import pandas as pd

# создаем словарь с данными
data = {
    "Имя": ["Аня", "Борис", "Вика"],
    "Возраст": [23, 30, 27],
    "Город": ["Москва", "СПб", "Казань"]
}

# создаем датафрейм
df = pd.DataFrame(data)
# печатаем, чтобы увидеть, что это такое
print(df)
```

``` output
     Имя  Возраст   Город
0    Аня       23  Москва
1  Борис       30     СПб
2   Вика       27  Казань
```

- Но зачем создавать новый [[Объект]], когда у нас и так есть [[тип данных]] **[[словарь]]**?
	- С датафреймами можно гораздо легче совершать операции, логичные для таблиц данных. Например, достать, вставить новую или удалить старую строку.
	- Если мы бы работали со словарем, а не с датафреймом, все эти операции пришлось реализовывать самостоятельно, а это долго и муторно!


Чтобы вывести содержимое столбца, нужно указать его название таким образом:

```python
print(df["Возраст"])
```

``` output
0    23
1    30
2    27
Name: Возраст, dtype: int64
```
