Определяет такие действия:
- r - чтение 
- w - запись 
- x - выполнение 
Права назначаются трём группами:
- пользователь
- членам группы-владельца 
- всем остальным пользователям

-rwxr-xr-x. 
    Первый символ (-) указывает, что это обычный файл.
    Следующие три символа (rwx) — права владельца.
    Следующие три (r-x) — права для группы.
    Последние три (r-x) — права для всех остальных.
