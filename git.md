Система контроля версий [[репозиторий|репозиториев]]
## команды 
- git clone <ссылка> - загрузка репозитория из ссылки на устройство
- git add <имя файла> - добавление файла в git
    - -a или . добавляет в git все файлы
- git commit -m <текст> - сохраняет изменения и добавляет в историю репозитория 
- git push - отправить изменения на сервер
- git pull - обновить [[локальный репозиторий]]
