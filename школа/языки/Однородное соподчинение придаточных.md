 [[Сложносочинённые предложения (ССП)]]
- от нескольких частей предложения задаётся одинаковый вопрос к одному слову в одной части
- принадлежат к одному [[типы придаточных частей в ССП|типу]]
- `[В пензенской области есть место], какое? (где родился великий поэт) и какое? (где он провёл своё детство)`
1. между придаточными НЕТ союза => запятая между частями
2. одиночный союз `и, или, либо` => НЕТ запятой (союз может быть опущен)
3. части между [[Противительные союзы]] `но, однако, а` => запятая
4. союзы повторяются => запятая
