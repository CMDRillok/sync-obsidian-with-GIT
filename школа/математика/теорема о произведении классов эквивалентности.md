- доказательство

	- n * p = (lk + r1)(qk + r2) = lkqk + lkr2 + r1qk + r1r2 = (lqk + lr2 + r1q) * k + r1r2
	-  пусть остаток от деления на K чиста r1r2 = r, то есть r1r2 = sk + r. 
	- тогда np = tk + sk + r = (t + s) * k + r. 
	- значит r - остаток от деления на k числа np
