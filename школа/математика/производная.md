**Производная функции** показывает, как функция меняется в точке ([[точка]]).
Давай рассмотрим функцию — скорость движения человека. Тогда производная от этой функции — это его ускорение.
А вот два графика скоростей человечков:
![[Pasted image 20250804181059.webp]]

Что такое [[ускорение]]? Это изменение скорости: как изменилось f(x) за небольшое изменение x.
Скорость первого человечка не меняется. Ускорение (производная) всегда равно нулю.
А вот скорость второго — равномерно вырастает: за одну единицу времени скорость возрастает на 1. Значит, ускорение (производная) равно 1 / 1 = 1.

Давай рассмотрим функцию посложнее: ([[квадратное уравнение]], [[квадратичная функция]])
![[Pasted image 20250804181229.webp|458]]
Как посчитать ускорение? Посмотрим, как менялась скорость за первую секунду (на отрезке (0, 1)) и за последнюю (на отрезке (3, 4)):
![[Pasted image 20250804181305.webp|459]]
Наклон прямых, проведенных через точки, показывает изменение скорости:
![[Pasted image 20250804181319.webp]]
Это и есть производные! Посчитать их примерные значения в выбранных областях мы можем, посчитав отношение Δy к Δx.
Ускорение в начале графика — это 1 / 1 = 1. Зато ускорение в конце — это (16 - 9) / (4 - 3) = 7 / 1 = 7!
А для достаточно маленького Δx **производная будет точно показывать направление функции вокруг выбранной точки.**
То есть, с помощью подсчета производной мы можем вычислять, в каком направлении идти при градиентном спуске!

Определим производную функции `y = 3 * x`. Для этого нужно посчитать соотношение Δy к Δx. Но чему равно Δy?
![[Pasted image 20250804181603.webp|589]]
Значит, соотношение Δy к Δx равно 3!
Для линейной функции `y = kx` производная будет равна k.
![[Pasted image 20250804181641.webp]]

[[sympy]]
```python
import sympy as sp # импортируем библиотеку

# объявляем переменную функции
x = sp.symbols('x')

# задаем функцию
f = 3*x + 7

# находим производную
df_dx = sp.diff(f, x)

print(df_dx) # напечатает 3
```
