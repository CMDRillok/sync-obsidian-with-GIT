с примера
	H<sub>2</sub> + N<sub>2</sub> = NH<sub>3</sub> + Q - [[обратимые реакции|обратимая]]
	прямая - соединение, экзотермическая, ОВР
	обратная - разложение, эндотермическая, ОВР
1) V<sub>пр</sub> = max
   V<sub>об</sub> = 0
2) V<sub>пр</sub> = становится **меньше**
   V<sub>об</sub> = становится **больше**
3) V<sub>пр</sub> = V<sub>об</sub>  - **Установилось равновесие**
аммиак постоянно раскладывается на компоненты, а компоненты снова объединяются

**химическое равновесие** - это состояние при котором [[скорость химической реакции|скорость]] прямой реакции = скорости обратной реакции
![[принцип Ле-Шателье]]
на примере аммиак
	температура больше => реакция обратная (чтобы понизить температуру)
	температура меньше => реакция прямая (соединение [[тепловой эффект реакции|выделяет тепло]])
	концентрация H<sub>2</sub> больше, => реакция прямая 
	концентрация NH<sub>3</sub> больше, => реакция обратная
	концентрация N<sub>2</sub> меньше, => реакция обратная
